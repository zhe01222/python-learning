# APCS Python 衝刺筆記 07：矩陣轉換 (105年3月實作二 / ZeroJudge b965)

這題是 APCS 實作題非常經典的二維陣列魔王題。這題不只考驗陣列操作，更考驗「反向推導」的邏輯。

## 🚨 核心破解：時光倒流機制
題目給的是「轉換後的結果 B」，要求我們推回「原始矩陣 A」。所以必須啟動時光機：
1. **指令順序要顛倒**：原本的測資操作順序必須反過來。
   * Python 實作：`operations = operations[::-1]`
2. **動作本身要反向**：
   * 原本的「上下翻轉 (1)」，反動作依然是 **「上下翻轉」**。
   * 原本的「順時針旋轉 90 度 (0)」，反動作變成 **「逆時針旋轉 90 度」**。
