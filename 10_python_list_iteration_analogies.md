# Python 專題：List 走訪的兩大流派 (置物櫃與 osu! 分數比喻)

在 Python 中處理 List（陣列）時，最核心的技巧就是如何用 `for` 迴圈把裡面的東西拿出來。根據「是否需要位置資訊」，可以分為以下兩種寫法。

---

## 🧱 基礎觀念：置物櫃比喻
想像有一排置物櫃 `a = [1, 6, 3, 5, 4, 3, 1]`：
* **Index (索引值)**：櫃子門上的「編號」（從 **0** 開始）。
* **Value (成員值)**：櫃子門打開後，裡面裝的「數字」。



---

## 🎧 實戰比喻：osu! 分數記錄
假設一場比賽中，前 5 顆音符的判定分數為：`scores = [300, 100, 50, 0, 300]`

### 流派一：需要位置的「抓錯員」 (Index Approach)
當你需要知道「是哪一顆音符出錯」或是要比較「前後音符關係」時使用。

```python
scores = [300, 100, 50, 0, 300]

# 使用 range(len()) 產生編號：0, 1, 2, 3, 4
for i in range(len(scores)):
    # i 是「音符編號」
    # scores[i] 是「那顆音符的分數」
    print(f"第 {i} 顆音符分數是：{scores[i]}")
    
    # 範例應用：如果分數是 0，就能精確指出是第幾顆斷了 Combo
    if scores[i] == 0:
        print(f"抓到了！第 {i} 顆音符 Miss！")
