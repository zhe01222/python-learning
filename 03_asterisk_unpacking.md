# APCS Python 衝刺筆記：星號 (*) 解包運算子魔法

在 Python 中，星號 `*` 是一個非常實用的工具，它的核心概念是**「脫掉最外層的括號，把裡面的東西倒出來」**。這在處理二維陣列或輸出格式時超級好用。

## 1. 搭配 zip() 處理矩陣轉置
當我們要把一個二維矩陣交給 `zip()` 處理時，必須用 `*` 把矩陣「解開」，否則 `zip()` 會把整個矩陣當成單一物品，無法進行配對。

```python
matrix = [
    [1, 2, 3],
    [4, 5, 6]
]

# 錯誤寫法：zip(matrix) -> zip 只收到一個大包裹
# 正確寫法：zip(*matrix) -> zip 收到 [1, 2, 3] 和 [4, 5, 6] 兩個包裹，然後把它們像拉鍊一樣扣起來

for row in zip(*matrix):
    print(row)
# 輸出：
# (1, 4)
# (2, 5)
# (3, 6)
